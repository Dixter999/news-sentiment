# Task: Project Setup and Dependencies

## Description
Create the project structure, configure `pyproject.toml` with Playwright and Gemini dependencies, create `.env.example`, and set up the Python development environment. Install Playwright browser.

## TDD Requirements
**This project uses Test-Driven Development. You MUST:**
1. RED: Write failing test first (test project can be imported)
2. GREEN: Write minimal code to make test pass
3. REFACTOR: Clean up code while keeping tests green

See `.claude/rules/tdd.enforcement.md` for complete requirements.

## Acceptance Criteria
- [ ] Project structure matches PRD specification
- [ ] `pyproject.toml` with correct dependencies (playwright, google-generativeai, sqlalchemy, etc.)
- [ ] `.env.example` with DB and Gemini API key placeholders
- [ ] All `__init__.py` files created
- [ ] `pip install -e ".[dev]"` succeeds
- [ ] `playwright install chromium` succeeds
